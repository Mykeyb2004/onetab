# MVP Release Checklist

## 1. 代码与构建

- [ ] `npm run lint` 通过
- [ ] `npm run typecheck` 通过
- [ ] `npm run test` 通过
- [ ] `npm run build` 通过
- [ ] `npm run test:e2e` 通过或明确说明跳过范围
- [ ] `dist/manifest.json` 可加载

## 2. 功能回归

- [ ] 收纳当前窗口可用
- [ ] 收纳当前标签可用
- [ ] 收纳选中标签可用
- [ ] 整组恢复默认在新建窗口打开
- [ ] 单标签恢复后从原组移除
- [ ] 恢复策略开关生效
- [ ] 搜索支持组名、标签标题、URL
- [ ] JSON 导入导出可用
- [ ] 文本导入会跳过非法 URL 并提示数量
- [ ] 单组导出不会带入其他组数据

## 3. 权限与隐私

- [ ] Manifest 权限只有 MVP 必需项
- [ ] 设置页明确说明本地存储和无痕限制
- [ ] 未引入服务端依赖
- [ ] 不支持页面会显示清晰反馈

## 4. 文档

- [ ] PRD 最新
- [ ] 开发计划最新
- [ ] ADR 最新
- [ ] 架构设计文档与代码一致
- [ ] 测试策略与测试矩阵最新
- [ ] 本地开发 runbook 可执行

## 5. 手动 smoke test

- [ ] 安装扩展成功
- [ ] Popup 可打开
- [ ] Manager 可打开
- [ ] Options 可打开
- [ ] 背景 service worker 无启动报错

## 6. Chrome Web Store 准备

- [ ] 扩展名称、描述、版本号正确
- [ ] 截图与说明文案准备完成
- [ ] 权限说明与隐私说明准备完成
- [ ] 首版已知限制写清

## 7. 回滚准备

- [ ] 保留上一可用构建产物
- [ ] 如涉及 schema 变更，已有 migration 与兼容说明
- [ ] 如发现严重恢复问题，可快速下线当前包并回到上一版本
